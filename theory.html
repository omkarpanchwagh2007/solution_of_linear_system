Theory of Linear Systems

A system of linear equations is a set of one or more linear equations involving the same set of variables. The equations can be written as
Ax = B, where A is a coefficient matrix, x is a column vector of unknowns, and B is a column vector of constants.

  • A = Coefficient matrix of size m × n
  • x = Column vector of unknowns [x₁, x₂, ..., xₙ]
  • B = Column vector of constants [b₁, b₂, ..., bₘ]

Solutions to linear systems can be classified as:

Types of Solutions:

  • Unique Solution: When the system has exactly one solution (consistent and independent)
  • No Solution: When the system has no solution (inconsistent)
  • Infinitely Many Solutions: When the system has an infinite number of solutions (consistent but dependent)

Rank of a Matrix

The rank of a matrix is the maximum number of linearly independent rows (or columns) in the matrix.

Definition:

A natural number "r" is said to be rank of matrix A if and only if:

  • There exists at least one non-zero minor of order r in A
  • All minors of order (r+1) in A are zero, if they exist

Row Echelon Form

A matrix is in row echelon form if it satisfies two conditions:

Conditions for Row Echelon Form:

  • First 'r' rows are non-zero and remaining rows are zero (if exist).
  • In first 'r' non-zero rows, first element of each row is to the right of the first element of the preceding row.

Example of Row Echelon Form:

1234
0567
0089

Gaussian Elimination Method

Gaussian elimination is a systematic method for solving systems of linear equations. It involves three types of elementary row operations:

Elementary Row Operations:

  • Swapping: Interchanging two rows
  • Scaling: Multiplying a row by a non-zero constant
  • Row Addition: Adding a multiple of one row to another row

The goal of Gaussian elimination is to transform the augmented matrix [A|B] into row-echelon form or reduced row-echelon form through these operations.
