Theory of Linear Systems

A system of linear equations is a collection of one or more linear equations involving the same set of variables. For example, a system with m equations and n unknowns can be written in matrix form as Ax = B, where A is an m × n coefficient matrix, x is an n × 1 column vector of unknowns, and B is an m × 1 column vector of constants.

a₁₁x₁ + a₁₂x₂ + ... + a₁ₙxₙ = b₁
a₂₁x₁ + a₂₂x₂ + ... + a₂ₙxₙ = b₂
⋮
aₘ₁x₁ + aₘ₂x₂ + ... + aₘₙxₙ = bₘ

Solutions to linear systems can be classified as:

Types of Solutions:

  • Unique Solution: When the system has exactly one solution (consistent and independent)
  • No Solution: When the system has no solution (inconsistent)
  • Infinitely Many Solutions: When the system has an infinite number of solutions (consistent but dependent)

Gaussian Elimination Method

Gaussian elimination is a systematic method for solving systems of linear equations. It involves three types of elementary row operations:

Elementary Row Operations:

  • Swapping: Interchanging two rows
  • Scaling: Multiplying a row by a non-zero constant
  • Row Addition: Adding a multiple of one row to another row

The goal of Gaussian elimination is to transform the augmented matrix [A|B] into row-echelon form or reduced row-echelon form through these operations.

Partial Pivoting

Partial pivoting is a technique used to improve numerical stability by selecting the largest absolute value element in the current column as the pivot element. This helps reduce rounding errors when solving systems with floating-point arithmetic.

Geometric Interpretation

Linear systems have intuitive geometric interpretations:

  • In 2D, each equation represents a line. The solution is the intersection point(s)
  • In 3D, each equation represents a plane. The solution is the intersection line(s) or point
  • Matrix transformations can be visualized as vector field operations

For a 2×2 system:

a₁₁x + a₁₂y = b₁
a₂₁x + a₂₂y = b₂

The solution corresponds to the intersection point of the two lines represented by these equations. If the lines are parallel (no intersection), the system has no solution. If the lines coincide (identical), the system has infinitely many solutions.

Historical Context

The method of Gaussian elimination is named after the German mathematician Carl Friedrich Gauss, though similar methods were known to Chinese mathematicians as early as 179 CE.

Applications

Linear systems are fundamental in:

  • Circuit Analysis
  • Structural Engineering
  • Computer Graphics
  • Economics Modeling
  • Machine Learning

Important Notes

• Gaussian elimination requires O(n³) operations for an n×n system

• Partial pivoting enhances numerical stability

• Integer inputs yield exact integer solutions when possible
